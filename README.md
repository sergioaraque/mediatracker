# MediaLog

Catálogo personal para gestionar tus **películas**, **series** y **libros** en un único lugar.

## Qué ofrece

- Añade películas, series y libros con portada, género, año y sinopsis
- Valora cada título del 1 al 10
- Estados por contenido: **Viendo**, **Pendiente** y **Visto**
- Seguimiento de temporadas y episodios para series
- Filtros por tipo, estado y búsqueda por título o género
- Ordenación por fecha, título, año o valoración
- Acceso privado: cada usuario solo ve su propio catálogo

## Tecnologías

| Capa | Tecnología |
|---|---|
| Frontend | Vue 3 + Vite + TypeScript |
| Estilos | Tailwind CSS v3 |
| Estado | Pinia |
| Enrutado | Vue Router 4 |
| Iconos | Lucide Vue Next |
| Backend / Auth / BD | Appwrite (self-hosted) |
| Servidor estático | nginx |
| Contenedor | Docker + Docker Compose |
| PWA | Service Worker + Web Manifest |
